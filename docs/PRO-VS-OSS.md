# OQA System: Open Source vs Pro

## Open Source (Free)

- Full quiz engine with MCQ + True/False
- Stratified random question groups
- Anti-cheat logging + live monitoring
- CSV export
- Docker + self-host ready
- All current features in this repo

## Pro Version (Paid / Freelance)

Contact `contact@your-freelance-domain.com` for pricing and demos.

**Additional capabilities typically included:**
- Hosted SaaS with zero maintenance
- Advanced per-student + class analytics dashboard
- Bulk student import (CSV + Google Classroom sync)
- Custom branding and white label
- Written answer / calculation question type with manual grading UI
- Detailed PDF reports + certificates
- Role-based access for school admins
- Audit logs and compliance exports
- Priority support + SLA
- Custom feature development

The open source version is production ready for most individual teachers and small schools.

For institutions wanting more polish, support, and time savings, the Pro offering is the recommended path.
